Ques-01: Discuss the scope of var, let, and const.

Ans:
We know that there are two types of scopes in Java Scripts. One is functional and another is block. Var is globally scoped or functional scoped while let and const are block scoped. Var variables can be updated and re-declared within its scope; let variables can be updated but not re-declared; const variables can neither be updated nor re-declared. Block scoped variables can not be accessed outside the blocks but var scoped variables can.

Ques-02: Tell us the use cases of null and undefined.

Ans:
Undefined occurs when a variable has been declared, but its value has not been assigned. And when function parameters that are not provided with values are also undefined. Null means an empty value or a blank value. It is used to empty the value of an object.

Ques-03: What do you mean by REST API?

Ans:
REST API stands for Representational State Transfer Application Programming Interface. It is a collection of rules that implies how applications or devices can connect to and communicate with each other. It uses all http methods to communicate.
